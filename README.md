#  ShowTime - Movie Ticket Booking System

A Django-based web app for booking movie tickets with dynamic pricing and OTP authentication.

##  Features
- OTP Login (Email)
- Movie Listing (OMDB API)
- Show Timings
- Seat Booking System
- Dynamic Pricing
- Booking History
- Razorpay Integration (in progress)

##  Tech Stack
- Django
- HTML, CSS
- SQLite/PostgreSQL
- SendGrid
